Colme Demo

Thank you for your interest in this plugin. In this demo there are three
tables for your to play with. The first one is very large. The second one
is inside a container. The third one doesn't have anything special.

You can try to move columns by dragging the table headers. All columns in
that column group will move together.

You can resize columns or columns groups using the little bar in the header
cells. Again, resizing a column group resizes all columns in the group.

Use the buttons above each table to toggle the visibility of the columns.
Hiding a column also hides its descendants. Showing a column also shows
all ancestors.

The plugin allows you to save the layouts (for instance if you want to
save them in a database). Press "Add Layout" to save the current value
and then press the newly created layout button to restore that layout.
Note: the buttons are not a feature, but an example of a possible use.
